A Collection of Design Work

I aim to create modern, clean designs whether it's for web or print. I enjoy putting a modern twist on my designs, experimenting with minimalism, overlaying elements, typographic styling and exploring a variety of illustration styles.

This collections includes a bunch of design experiments and side projects including: UI, Illustration, Print Design and Graphic Design.

Elements

The Elements project is an educational tool to help children learn about the Periodic Table. The idea is that this would be used as a revision tool for children aged 9-12.
Given the young age group of users, I used this project as a chance to experiment with a bright colour palette, CSS animations and illustrations.

Wanderlust: Icon Set

Wanderlust is an illustrative icon set featuring some of the top European landmarks. I also designed a website to feature the icons on.

When I set out to create this icon set, I wanted it to differ from anything I had created before. So, I decided to create a highly detailed icon which not only was instantly recognisable to the user, but that the smaller details of the landmark could be seen when zoomed up on.
